## ✨ Features
- **相互連携**\
MisskeyインスタンスはMastodon や Pixelfed などの他のネットワーク上のユーザーと友達になることもできます
- **リアクション**\
Misskeyは絵文字リアクションに使えます
- **ドライブ**\
クラウドを搭載しているので投稿画像を貯蔵できます
- **高機能なウェブUI**\
	UIを自由にカスタマイズできます。
- And much more...
